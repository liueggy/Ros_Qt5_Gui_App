# Ros_Qt5_Gui_App Windows Latest Binaries

这是 GitHub Actions 自动生成的 Windows 可运行目录，来源于源码分支 $env:GITHUB_REF_NAME。

## 第一次使用

`at
git clone --depth=1 --branch windows-latest-bin https://github.com/liueggy/Ros_Qt5_Gui_App.git RosQt5Latest
cd RosQt5Latest
update_and_run.bat
`

## 以后更新并运行

双击：

`at
update_and_run.bat
`

或命令行执行：

`at
git fetch --depth=1 origin windows-latest-bin
git reset --hard origin/windows-latest-bin
run.bat
`
